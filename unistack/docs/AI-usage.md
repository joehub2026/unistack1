# AI Usage Disclosure — UniStack

## Policy
This project used AI tools as a **helper and learning accelerator**, not as a replacement
for team understanding, decision-making, or coding effort.

## How AI Was Used
| Task | AI Used? | Notes |
|---|---|---|
| Brainstorming features | ✅ Yes | Generated initial ideas; team selected and refined |
| Writing user stories | ✅ Yes | Draft generated; team reviewed and adapted to INES context |
| Code generation | ✅ Partially | AI suggested patterns; all code reviewed, understood, and modified by team |
| Database schema design | ❌ No | Designed by team from scratch based on requirements |
| Testing | ❌ No | All tests run manually by team members |
| Debugging | ✅ Partially | AI helped explain error messages |
| Final decision-making | ❌ No | All architectural and design decisions made by team |

## Team Commitment
Every member of this group has read, understood, and is able to explain every file in
this repository. AI was used as a tool — like documentation or Stack Overflow — not as
an author.
